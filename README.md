# 💰 FinFlow — Personal Finance Tracker

FinFlow is a full-stack web application that helps users take control of their personal finances. Track your income and expenses, set monthly budgets, monitor savings goals, and analyze spending trends — all in one place.

---

## ✨ Features

- **Dashboard** — monthly summary with income, expenses, balance and savings rate
- **Transactions** — add, edit, delete and search transactions with CSV export
- **Budgets** — set spending limits per category with real-time progress tracking
- **Savings Goals** — create goals and track progress toward each target
- **Reports** — 12-month balance trend, income vs expenses charts, top spending categories
- **Authentication** — secure registration and login with JWT
- **Multi-currency** — switch between UAH, USD, EUR and GBP
- **Fully responsive** — works on desktop, tablet and mobile
